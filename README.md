# The Banker

A website on loaning money.

## Overview

The Banker is a financial web application designed to help users understand and manage loan processes. This project provides an interactive platform for loan information and calculations.

## Tech Stack

- **HTML** - 75.5% (Markup structure)
- **JavaScript** - 22.6% (Interactive functionality)
- **CSS** - 1.9% (Styling)

## Features

- Loan information resources
- Interactive loan calculators
- User-friendly interface
- Responsive design

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Yolanda-loo/The-Banker.github.io.git
   ```

2. Open `index.html` in your web browser to view the website.
